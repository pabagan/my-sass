# Text states
```sass
/* Font families  */
.is_mainfont{ font-family: $main-font}
.is_secondfont{ font-family: $second-font}

/* Weight */
// ---------------------------
.w100{ font-weight: 100 !important }
.w200{ font-weight: 200 !important }
.w300{ font-weight: 300 !important }
.w400{ font-weight: 400 !important }
.w600{ font-weight: 600 !important }
.w700{ font-weight: 700 !important }
.w800{ font-weight: 800 !important }
.w900{ font-weight: 900 !important }

/* Transform */
// http://www.w3schools.com/cssref/pr_text_text-transform.asp
// ---------------------------
.is_upper{
    text-transform: uppercase;
    //letter-spacing: 0.4px;
}
.is_lower{text-transform: lowercase}
.is_caps{text-transform: capitalize}


/* Font Style */
// ---------------------------
.is_regular { font-style: normal }
.is_oblique { font-style: oblique }
.is_italic{ font-style: italic }
```
