# Grid

## Width grid
12 column width grid with `col-[xs,sm,md,lg]-[1-12]`.

```html
<div class="row">
  <div class="container">
    <p>Hi there human!</p>
  </div>
  <div class="row-xs-12 row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="row-xs-12 row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="row-xs-12 row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="row-xs-12 row-lg-6">
    <h3>Grid width content</h3>
  </div>
</div>
```
